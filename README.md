# Bar and Restaurant Management System

A simple Python + MySQL CRUD project to manage menu, customers, and orders for a bar & restaurant.

## Features
- Manage menu items (add, update, delete)
- Manage customers
- Record and view customer orders
- Calculates total price automatically

## Technologies Used
- Python
- MySQL
- MySQL Connector

## Setup
1. Import `bar_restaurant.sql` into your MySQL server.
2. Edit database credentials in `bar_restaurant.py` if needed.
3. Run:
   ```bash
   python bar_restaurant.py
